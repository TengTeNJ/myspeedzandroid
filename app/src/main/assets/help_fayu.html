Introduction

La présente APP est conçue pour être utilisée avec « Myspeedz ». Elle reçoit les données de mesure de la vitesse à partir de notre « Myspeedz », enregistre les données des différents utilisateurs et affiche l’historique des données sous forme de graphique ou d’autres formats.


Top

Appareils supportés

Systèmes d’exploitation IOS avec Bluetooth 4.0 et plus.


Top

Données sauvegardées dans l’APP

Données de mesure：« Myspeedz » transfère les données de mesure dans l’APP。

Profil d’utilisateur：Suivant la vitesse sauvegardée par un utilisateur. Vous avez la possibilité de créer de nouveaux profils d’utilisateurs via le point de menu Configuration.

 

Lors de la première mise en marche de l’App, plusieurs exemples de profils d’utilisateurs seront crées. Vous pourrez vous exercer à travailler sur les données d’utilisateurs à partir de ces comptes.

Configuration initiale

Utilisation de l’App pour recevoir les données de notre « Myspeedz » pour la première fois. Nous vous conseillons de suivre les 2 étapes suivantes :


1、Activez le Bluetooth de votre appareil et connectez-vous sur « Myspeedz »。
2、Cliquez sur le bouton de Configuration (sur la barre d’outils en bas de l’écran),     

Cliquez ensuite sur Ajouter un nouvel utilisateur, et saisissez votre nom/ID.


Top

Réception des données

L’APP et « Myspeedz » communiquent par Bluetooth. Pour activer le Bluetooth sur l’appareil, allez dans les réglages et recherchez « Myspeedz ». Procédez enfin à l’appariement.


Réception des données de vitesse：

L’App doit être en marche
Retour au menu principal de l’APP
Activez « Myspeedz »

Faire une simulation.

Transfert automatique des données de vitesse. L’écran d’App sera mis à jour par rapport aux données reçues.

Touchez et glissez vers le milieu de l’écran pour enregistrer les données de vitesse.


Top

Historique de données de navigation

Vous pouvez visualiser les données de navigation sous forme de tableau ou de graphique.


Sélectionnez premièrement l’utilisateur souhaité en glissant le milieu du menu principal. Cliquez ensuite sur GRAPHIQUE ou LISTE sur la barre d’outils en bas de l’écran.

Fonctions disponibles dans le Graphique:

Les courbes différentes sont affichées en différentes couleurs. La courbe correspondante est affichée en étiquette en haut avec la même couleur de fond. 

Cliquez dans la zone du graphique pour afficher les valeurs de détail pour chaque mesure.
Cliquez sur l’étiquette du graphique de la courbe pour afficher ou masquer la courbe correspondante.


Liste des fonctions:

Les données de vitesse sont listées à chaque fois
Cliquez sur « par vitesse » pour afficher les détails ou supprimer.
Cliquez sur Supprimer Tout en bas de l’écran pour effacer toutes les vitesses d’un utilisateur.

Gestion des utilisateurs

Modifier un profil d’utilisateur

Cliquez sur le bouton Configuration en bas de la barre d’outils dans le menu principal. Sélectionnez un profil d’utilisateur existant pour le modifier ou le supprimer ou simplement pour AJOUTER un nouvel utilisateur.

Dépannage et FAQ

1、Je n’arrive pas à recevoir les données de « Myspeedz »
Assurez-vous que le Bluetooth est activé et qu’aucun autre appareil ne fonctionne avec la même App. Veuillez utiliser séparément. 
2、Le message « La plateforme/le matériel ne supporte pas le Bluetooth » s’affiche.
L’App fonctionne avec le Bluetooth 4.0 ou plus. Veuillez connecter un appareil compatible pour recevoir les données de vitesse.
3、Pourrais-je rentrer les données manuellement ?
Non. Les données peuvent uniquement être reçues de l’APP de « Myspeedz ».


Les données affichées par « Myspeedz » et APPS le sont uniquement à titre de référence et n’engagent aucunement sa responsabilité.
