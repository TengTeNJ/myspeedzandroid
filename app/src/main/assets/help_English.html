Introduction

This APP is designed to use with “Myspeedz". It receives measurement data of speed from our “Myspeedz”, store data with different users, and display historical data in graph or other formats.


Top

Supported Devices

Devices with IOS 9.0 or above, and with Bluetooth 4.0 or above.


Top

Data Stored in the APP

Measurement data：“Myspeedz” transfers the measurement data to the APP。

User profile：Per speed stored by a user. You can create new user profiles via the Setup menu item.

 

When the App is running for the first time, several example users’ profiles will be created. You can get experiences on working with user data by those examples.

Initial Setup

Using the App to receive data from our “Myspeedz” for the first time, please follow up 2 steps as:


1、Turn on your Bluetooth on your device and connect to “Myspeedz”。
2、Click the Setup button (at the toolbar at the bottom of the screen),     

then click Add a new user, and input your name/ID there.


Top

Receiving data

The App and the “Myspeedz” communicate via Bluetooth. Firstly, turn on Bluetooth on the iOS device in the settings page.Then open APP named “Myspeedz”, click the button named “Device” in the upper right corner of the home page, and then select the "Myspeedz" device in the pop-up Bluetooth device list to connect.


Receiving data of speed：

The App should be running
Go back to the APP’s main menu
Turn on the “Myspeedz”

Practising or making a shot.

Automatically transfer the data of speed, the App screen will be updated to the received data, and the color of screen turning to green.

Touch and swipe at the middle part of the screen to save the data of speed.


Top

Browsing data History

You can browse historical data in a chart format or a tabular format.


First, select the desired user by swiping in the middle part of the main menu. Then, click either CHART or LIST from the bottom toolbar.

Functions available in Chart:

Different curves are shown in different colors. The corresponding curve is shown in a label at the top with a same color background. 

Click on the chart area to show the detail values per each measurement.
Click the curve graph label to show or hide the corresponding curve.


Functions List:

Speed data are listing per each time
Click on per speed to view its details, or delete.
Click Delete All at the bottom of the screen to delete all speeds with that user.

User Management

Modify a user profile

Click Setup button at the bottom toolbar in the main menu.

In the main menu, click 'KPH' or 'MPH' to switch.

Trouble Shooting and FAQ

1、If I cannot receive data from “Myspeedz”
Make sure Bluetooth is turned on, and no other device is running the same APP, keeping in using separately. 
2、The message “The platform/hardware not support Bluetooth”
The App must be running on a Bluetooth(4.0 or above) enabled device in order to receive data of speed.
3、May I manually input data?
No, Data can only be received from a APP of “Myspeedz”.
